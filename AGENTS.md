# AGENTS.md - Issue Discovery Orbit

你在 Issue Discovery Orbit 中工作。这个 orbit 用于把 PRD synthesis 和 issue slicing 转成 issue tracker 中的可追踪工作。

## 必须遵守

1. 执行本 orbit 的任何 skill 前，先读 `docs/issue-discovery-orbit/discovery-rules.md`。
2. 发布 PRD 或 issues 时按目标仓库 Repository Publishing Rules；规则缺失或冲突时，只输出 candidates。
